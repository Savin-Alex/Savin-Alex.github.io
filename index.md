A privacy-first, real-time AI assistant for conversations, interviews, meetings, and educational sessions.

### Multi-Agent AI Development Pipeline

MVP-пайплайн, который проводит небольшую задачу через Jira, планирование, генерацию кода, ревью и Pull Request в GitHub.

- Поэтапный MVP, где каждая фаза даёт самостоятельный результат.
- Объединяет роли planner, coder и reviewer с Jira и GitHub в один управляемый цикл.
- Измеримые результаты: количество итераций, стоимость, доля успешных задач, экономия lead time.

Stack: Python, Claude API, Jira Cloud, GitHub API, pytest

[View project page](projects/ai-dev-pipeline.md)
